문제 14번: 상관 서브쿼리의 이해

상관 서브쿼리는 메인 쿼리의 값을 서브쿼리가 참조하여 실행되는 구조입니다. 성능 최적화를 위해서는 실행 계획을 반드시 확인해야 합니다.

SELECT a.EMP_NAME, a.SALARY
FROM EMP a
WHERE a.SALARY > (
    SELECT AVG(b.SALARY)
    FROM EMP b
    WHERE b.DEPT_ID = a.DEPT_ID
);

위 쿼리는 각 부서별 평균 급여보다 많이 받는 사원을 추출하는 예시입니다.
