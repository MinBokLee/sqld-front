## 🛠️ 상세 수정 내역

### 1. 인증 시스템 안정화
- **api.ts (인터셉터) 수정**:
    - 로그인 API(`signIn`) 호출 시 발생하는 401 에러를 토큰 갱신(`token-refresh`) 로직에서 예외 처리.
    - 이를 통해 잘못된 로그인 정보 입력 시 서버의 실제 에러 메시지("아이디 또는 비밀번호가 일치하지 않습니다")가 정상 출력되도록 보장.
- **UserContext.tsx 수정**:
    - `login` 함수 실패(`catch`) 시 무조건 호출되던 `clearUser()` 제거.
    - 로그인 시도 자체가 실패한 경우 기존 세션을 보존하여 UI 상태 불일치 방지.

### 2. 오픈채팅(Open Chat) 기능 고도화
- **ChatContext.tsx (데이터 흐름) 개선**:
    - 주석 처리되어 있던 `fetchChatHistory()`를 활성화하여 입장 시 최근 대화 및 출입 기록 동기화.
    - 새로 들어온 사용자가 기존 대화 맥락과 현재 참여자(입장 메시지 기준)를 즉시 인지 가능.
- **OpenChat.tsx (UI/UX) 개선**:
    - **시스템 메시지 스타일링**: 입장(`ENTER`), 퇴장(`LEAVE`) 메시지에 전용 배경색, 블러 효과, `Users` 아이콘 적용.
    - **애니메이션 추가**: 시스템 메시지에 `fade-in` 및 `slide-in` 효과를 부여하여 출입 알림 인지도 강화.
    - **가독성 보정**: 메시지 수신 시 부드러운 스크롤 최적화 및 타임스탬프 스타일 정교화.
