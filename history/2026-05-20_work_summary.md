# 📅 작업 일지 (2026-05-20)

## 🛠️ 1. 환경 설정 및 네트워크 인프라 최적화
- **백엔드 포트 변경 반영:** 백엔드 서비스 포트가 8881에서 80으로 변경됨에 따라 관련 설정을 업데이트함.
- **Dev/Prod 환경 분리 완벽화:** 
  - `.env.development`, `.env.production`에서 `VITE_API_BASE_URL`을 비워 코드가 주소에 직접 의존하지 않도록 수정.
  - `vite.config.ts`에서 로컬 개발용 프록시 타겟을 `http://localhost:8080`(또는 운영IP)으로 고정 설정.
  - 이를 통해 로컬 환경과 운영 환경이 서로 겹치지 않고 각자의 백엔드를 올바르게 바라보도록 개선.
- **HTTPS 보안 오류(Mixed Content) 해결:** 
  - 운영 도메인 접속 시 발생하는 `ERR_CERT_COMMON_NAME_INVALID` 에러를 해결하기 위해 모든 API 호출을 상대 경로(`/api`)로 통일.
  - 브라우저의 HSTS 캐시 문제 원인 분석 및 해결 가이드 제공.
- **이미지 및 API 프록시 설정:** 
  - `public/_redirects` 파일을 생성하여 `/api/*` 및 `/uploads/*` 요청을 백엔드 서버로 리다이렉트하도록 설정. (이미지 404 및 보안 차단 문제 해결)

## 📱 2. 반응형 디자인 및 모바일(S25+) 최적화
- **팝업(모달) 시스템 개선:** 
  - 로그인, 회원가입, 시험 일정 모달에 `max-h-[90vh]` 및 `overflow-y-auto`를 적용하여 모바일에서 잘림 현상 해결.
  - 회원가입 및 일정 모달의 헤더를 `sticky`로 고정하여 스크롤 시에도 닫기 버튼 접근이 가능하도록 개선.
- **UI 가독성 향상:**
  - **Hero 섹션:** 제목 폰트 크기를 반응형(`text-3xl md:text-5xl`)으로 조정하고 패딩 최적화.
  - **게시판 상세:** 제목 폰트 크기를 단계적(`sm:text-3xl`, `lg:text-4xl`)으로 완화하고 줄 간격(`leading-snug`)을 넓혀 가독성 확보. 
  - **알림창:** 모바일에서 알림 드롭다운 너비가 화면을 넘지 않도록 가변 너비(`w-[calc(100vw-2rem)]`) 적용.
