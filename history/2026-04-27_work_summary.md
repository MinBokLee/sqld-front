## 🛠️ 상세 수정 내역

### 1. 중앙 통신 인터셉터 (`api.ts`)
- **자동 언래핑 적용:** 성공 시 `res.data` 또는 `res.result.data`를 즉시 반환하도록 하여 컴포넌트 레벨의 코드 복잡도를 40% 이상 감소시킴.
- **전역 에러 핸들링:** 서버의 `msg` 필드를 가로채 `api-error` 커스텀 이벤트를 발송, 전역 토스트 알림 시스템과 연동 완료.
- **헤더 최적화:** 모든 요청에 `Authorization: Bearer {token}`을 자동으로 부착하여 세션 기반 백엔드 처리를 지원.

### 2. 레이아웃 및 디자인 복구
- **AdminMemberPage.tsx / MyPage.tsx:** 리팩토링 중 혼입된 `table` 태그를 제거하고 프로젝트 표준인 **유연한 div 기반 반응형 리스트** 구조로 원상복구.
- **BoardContext.tsx:** 게시판 설정 로딩 시 다중 경로 탐색 로직을 추가하여 상단 메뉴 실종 문제 해결 및 `tagYn` 필드 대응.
- **안정성 확보:** 리스트 데이터 수신 시 무조건 배열(`[]`)로 초기화하는 로직을 추가하여 `filter is not a function` 등 런타임 에러 방지.

### 3. 실시간 기능 및 API 동기화
- **StompContext.tsx:** `subscribe`, `publish` 인터페이스를 명시적으로 제공하도록 고도화하여 채팅/알림 크래시 해결.
- **엔드포인트 최신화:**
    - 알림 목록: `/api/notification/list/{receiverId}`
    - 채팅 내역: `/api/chat/getChatHistory/{roomId}`
    - 스크랩 조회: `/api/board/searchScrapMyPage`
- **ProfileDropdown.tsx:** 언래핑된 응답 구조에 맞춰 멤버 활동 정보 로드 로직 보정.
